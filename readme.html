üìÑ API Endpoints

Method Endpoint Description Response
GET / Welcome 200 OK
GET /api-docs API Docs 200 OK
GET /incidents List incidents 200 OK
GET /incidents/:id Incident by ID 200 OK/404 Not Found
POST /incidents New incident 201 Created/400 Bad Request
DELETE /incidents/:id Delete incident 200 OK/404 Not Found
